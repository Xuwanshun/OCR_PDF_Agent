OCR PDF Agent

From Raw PDFs to Grounded Answers

A single-agent OCR + RAG workflow that extracts text and visual regions, stores local memory in Chroma, and answers questions with source-aware citations.

View Evaluation Report Read Project README

System Flow

Pipeline in 5 steps

1

Ingest

Read PDFs from Document/ and generate per-page images.

2

Extract

Use native PDF text, fallback OCR, and detect table/chart/figure regions.

3

Structure

Produce document.json, document.md, and metadata artifacts.

4

Index

Chunk and embed into local Chroma for retrieval.

5

Answer

Serve ocr ask / ocr chat with cited context.
